# Проекты по тестированию

Этот репозиторий содержит независимые проекты по автоматизации тестирования:

1. **Управление документами** — юнит-тесты логики работы с документами и полками.
2. **Тестирование API Яндекс.Диска** — интеграционные тесты с внешним API.
3. **Геометрические и математические задачи** — юнит-тесты для проверки корректности расчётов.

---

## 1. Управление документами (`app.py`, `test_app.py`)

### Описание
Программа для работы с документами: поиск, добавление, удаление, перемещение по полкам. Реализована с возможностью тестирования.

### Функции
- Поиск владельца документа по номеру
- Добавление и удаление документов
- Перемещение между полками
- Просмотр всех документов

### Тесты
- Написаны на `unittest`
- Проверяют корректность работы всех ключевых функций
- Используют изолированное состояние (`setUp`)

### Запуск тестов

`bash python test_app.py`


---

## 2. Тестирование API Яндекс.Диска (`test_Yandex_Disk.py`)

### Описание
Интеграционные тесты для API Яндекс.Диска: создание, проверка и удаление папок.

### Функции
- Создание папки на Яндекс.Диске
- Проверка её наличия в списке
- Автоматическая очистка после теста

### Требования
- Токен Яндекс.Диска (в `.env`)
- Установленные зависимости: `pytest`, `requests`, `python-dotenv`

### Файл `.env`

`env YA_DISK_TOKEN=ваш_токен_здесь`


### Запуск тестов

`bash pytest test_Yandex_Disk.py -v`

## 3. Математические и геометрические задачи

### 3.1. Проверка треугольника (`check_triangle.py`, `test_check_triangle.py`)

#### Описание
Определяет тип треугольника по трём сторонам: равносторонний, равнобедренный, разносторонний или «не существует».

#### Тесты
- Проверка всех типов треугольников
- Валидация входных данных (ноль, отрицательные числа, неравенство треугольника)
- Написаны на `unittest`

#### Запуск тестов

`bash python test_check_triangle.py`

---

### 3.2. Решение квадратного уравнения (`quadratic_equation.py`, `test_quadratic_equation.py`)

#### Описание
Решает квадратное уравнение \( ax^2 + bx + c = 0 \), возвращая корни или сообщение «корней нет».

#### Функции
- Вычисление дискриминанта
- Нахождение корней при \( D > 0 \), \( D = 0 \), \( D < 0 \)
- Защита от деления на ноль (\( a = 0 \))

#### Тесты
- Проверка двух корней, одного корня, отсутствия корней
- Обработка ошибок при \( a = 0 \)
- Написаны на `unittest`, без зависимости от `print()` — используют `return`

#### Запуск тестов

`bash python test_quadratic_equation.py`
