# Проекты по тестированию

Этот репозиторий содержит два независимых проекта по автоматизации тестирования:

1. **Управление документами** — юнит-тесты логики работы с документами и полками.
2. **Тестирование API Яндекс.Диска** — интеграционные тесты с внешним API.

---

## 1. Управление документами (`app.py`, `test_app.py`)

### Описание
Программа для работы с документами: поиск, добавление, удаление, перемещение по полкам. Реализована с возможностью тестирования.

### Функции
- Поиск владельца документа по номеру
- Добавление и удаление документов
- Перемещение между полками
- Просмотр всех документов

### Тесты
- Написаны на `unittest`
- Проверяют корректность работы всех ключевых функций
- Используют изолированное состояние (`setUp`)

### Запуск тестов

`bash python test_app.py`


---

## 2. Тестирование API Яндекс.Диска (`test_Yandex_Disk.py`)

### Описание
Интеграционные тесты для API Яндекс.Диска: создание, проверка и удаление папок.

### Функции
- Создание папки на Яндекс.Диске
- Проверка её наличия в списке
- Автоматическая очистка после теста

### Требования
- Токен Яндекс.Диска (в `.env`)
- Установленные зависимости: `pytest`, `requests`, `python-dotenv`

### Файл `.env`

`env YA_DISK_TOKEN=ваш_токен_здесь`


### Запуск тестов

`bash pytest test_Yandex_Disk.py -v`


---

## Общие требования
- Python 3.6+
- Установка зависимостей:

`bash pip install pytest requests python-dotenv`
